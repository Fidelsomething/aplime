# aplime
quickstart boilerplate for nodeJS + serverSide Authentication for SQL


installation:
npm install

edit /api/config/projectconfig.js with your parameters,
change the name of your users table on authentication.js, along with your table information. The key point of this "boilerplate" is providing the syntax guideline.

run with npm start.

Note: passport ends up not being used, although it was a first iteration. encryptation of password can be changed for a better one.

Project originated as a need to develop an authentication API for a diferent project
